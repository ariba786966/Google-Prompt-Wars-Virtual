# Google-Prompt-Wars-Virtual
# Medical app dashboard

# The vision: - In emergency situation, a deaf person is in a critical situation and needs help. Every second matters. This app is design with vision that this app will be pre-installed in hospital's systems, it will help deaf people to communicate with the medical staff and get the assistance they need. 

# The app will have the following features: -
1. ASL keyboard with different font size.
2. Dark theme is use to reduce strain on eyes. Thought, a light toggle is provided as well.
3. A camera feature is provided to capture the sign language and translate it to text in real time. 
4. Cloud-Native Architecture: Deployed using Google Cloud Run for 99.9% availability and global access.
5. Mobile-Responsive: Works just as well on a tablet or phone during emergency rounds.
6. A good UI/UX feature for both left and right hand people.


# Tech Stack
Frontend: HTML5, CSS3 (Custom variables for the Gen-Z theme), JavaScript.
Backend: Node.js
Infrastructure:- 
- Docker: Containerized for consistent performance.
- Google Cloud Build: Automated CI/CD pipeline.
- Google Cloud Run: Serverless deployment.
